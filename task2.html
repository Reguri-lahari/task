!DOCTYPE html

It is an instruction to the web browser, specifying the document type and version of HTML being used

Head

which stands for HyperText Markup Language, is the foundation of all web pages.

Title

The tag defines the title of an HTML page. It's placed within the section of the HTML

Body

The tag contains all the visible content of the page

Footer

The tag is used to define a footer for a document or a section within the document

br

The Line Break element

hr

The hr tag in HTML creates a horizontal rule
