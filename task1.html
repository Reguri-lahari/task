What is Internet?

The internet is a global network of interconnected computer networks.

What is Browser?

A browser is an application program that provides a way to look at and interact with all the information on the World Wide Web.

What is Domain?

The domain of a function is the set of values that we are allowed to plug into our function.

What is IP Address?

An IP address is a unique numerical identifier.

What is URL?

A URL is the unique web address of a specific resource on the interne.t

What is Bug?

A "bug" refers to an error, flaw, or defect in software code or system that causes it to behave unexpectedly or incorrectly.

What is Debug?

Debugging is the process of finding and fixing errors, often called "bugs," in software or hardware.

What is HTTPS?

HTTPS is a secure version of HTTP that uses encryption to protect the data transmitted between a web browser and a website.

What is HTTP?

Hypertext Transfer Protocol, is the foundation of data communication on the World Wide Web.

What is Markup Language?

A markup language is a system that uses special codes or tags to structure and format text in a document.

What is Programming Language?

A programming language is a set of instructions that tells a computer what to do. It allows humans to communicate with computers and create software, websites.

What is XHTML?

Extensible Hypertext Markup Language is a cross between a programming language HTML & XML.

What is HTML?

HyperText Markup Language, is the foundation of all web pages. It's the language used to structure and present content, like text, images
